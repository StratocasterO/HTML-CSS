Estructura básica de un documento HTML5

La estructura básica de un documento HTML5 es la siguiente: 

			

<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Document</title>
	</head>
	<body>
	</body>
</html>

La etiqueta head contiene toda la configuración del archivo: título e icono en la pestaña del navegador, conjunto de caracteres, etc. La etiqueta body contiene el cuerpo del documento.

Los comentarios en un documento HTML se escriben dentro de <!-- comentario -->.
Dentro de una etiqueta o en un documento CSS, se escriben dentro de /* comentario */.

HTML5 con Bootstrap

Bootrstrap es un recurso que sirve para maquetar con clases definidas en una hoja de estilos externa en lugar de con una hoja de estilos propia. Los estilos se implementan asignando las clases correspondientes a los objetos.

Container, container-fluid, row, col.

Ancho de las columnas con col-n (tienen que sumar 12).
Márgenes y paddings con m-n, mt-n, mb-n, p-n, pt-n, pb-n, etc. n va de 1 a 5, siendo equivalente a 4, 8, 16, 24 y 48 px respectivamente.

Se puede hacer una web adaptativa fijando el ancho de las columnas para cada tamaño de pantalla: extrasmall (sin sufijo), small (-sm), medium (-md), large (-lg) y extralarge (-xl). Los anchos de página correspondientes son menor de 576 y mayor de 576, 768, 992 1200 px respectivamente.
